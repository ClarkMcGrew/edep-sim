/generator/

Control of the particle generator.

Sub-directories

/generator/count/Control number of primary vertices per event.
/generator/kinematics/Control kinematics generators.
/generator/position/Control vertex position the generators.
/generator/time/Control vertex time the generators.

Commands

clear
add
combine
allowEmptyEvents
addFakeGeantino
allowPartialEvents

clear

Clear the current list of vertex generators.

Available at all Geant4 states.

add

Add a generator using the current factories.

Available at all Geant4 states.

combine [src] [dest] [relative]

Combine the previous two vertices into one.

Available at all Geant4 states.

Parameters
src type i
dest type i
relative type b Omittable : default value = false

allowEmptyEvents []

If true, then generate events even if they don't contain a vertex.
If true, then generate the final event even if it ran out of interactions in the input kinematics file.

Available at all Geant4 states.

Parameters
type b

addFakeGeantino []

If true, then guarantee a primary vertex by adding a geantino.

Available at all Geant4 states.

Parameters
type b

allowPartialEvents []

Available at all Geant4 states.

Parameters
type b
