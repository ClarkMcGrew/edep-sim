Process Table control commands.

Sub-directories

/process/em/General commands for EM processes.
/process/eLoss/Commands for energy loss processes.
/process/msc/Commands for EM scattering processes.
/process/gconv/Commands for EM gamma conversion BH5D model.
/process/dna/Commands for DNA processes.
/process/had/Control of general hadronic physics parameters and choices.
/process/optical/Commands related to the optical physics simulation engine.

list [type]

List up process names
list [type]
type: process type [all:for all processes]

Available at all Geant4 states.

Parameters
type type s Omittable : default value = all Parameter candidates : all NotDefined Transportation Electromagnetic Optical Hadronic Photolepton_hadron Decay General Parameterisation UserDefined

verbose [verbose]

Set Verbose Level for Process Table
verbose [level]
level: verbose level

Range : verbose >=0

Available Geant4 state(s) : PreInit Init Idle GeomClosed EventProc

Parameters
verbose type i Omittable : default value = 1

setVerbose [verbose] [type]

Set verbose level for processes
setVerbose level [type or name]
level: verbose level
name : process name
type : process type
[all] for all processes

Available Geant4 state(s) : Idle GeomClosed EventProc

Parameters
verbose type i
type type s Omittable : default value = all

Available Geant4 state(s) : Init Idle GeomClosed EventProc

Parameters
procName type s
particle type s Omittable : default value = all
