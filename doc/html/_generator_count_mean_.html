/generator/count/mean/

Control of the mean count factory.

Commands

Set the intensity for the events. This is only for documentation and is saved in the MC header.

Available at all Geant4 states.

Parameters
intensity type d

number [mean]

Set the mean number of events to generate.

Available at all Geant4 states.

mean type d
