direction [Px] [Py] [Pz]

Set momentum direction.
Direction needs not to be a unit vector.
Angular distribution type is set to planar.

Range : Px != 0 || Py != 0 || Pz != 0

Available at all Geant4 states.

Parameters
Px type d
Py type d
Pz type d

energy [Energy] [Unit]

Set kinetic energy.

Available at all Geant4 states.

Parameters
Energy type d
Unit type s Omittable : default value = GeV Parameter candidates : eV keV MeV GeV TeV PeV meV J electronvolt kiloelectronvolt megaelectronvolt gigaelectronvolt teraelectronvolt petaelectronvolt millielectronVolt joule

position [X] [Y] [Z] [Unit]

Set starting position of the particle for a Point like source.
Same effect as the two /gps/pos/type Point /gps/pos/centre commands.

Available at all Geant4 states.

Parameters
X type d
Y type d
Z type d
Unit type s Omittable : default value = cm Parameter candidates : pc km m cm mm um nm Ang fm parsec kilometer meter centimeter millimeter micrometer nanometer angstrom fermi

ion [Z] [A] [Q] [E]

Set properties of ion to be generated.
[usage] /gps/ion Z A Q E
Z:(int) AtomicNumber
A:(int) AtomicMass
Q:(int) Charge of Ion (in unit of e)
E:(double) Excitation energy (in keV)

Available at all Geant4 states.

Parameters
Z type i
A type i
Q type i Omittable : default value = 0
E type d Omittable : default value = 0.0

ionLvl [Z] [A] [Q] [Lvl]

Set properties of ion to be generated.
[usage] /gps/ion Z A Q Lvl
Z:(int) AtomicNumber
A:(int) AtomicMass
Q:(int) Charge of Ion (in unit of e)
Lvl:(int) Number of metastable state excitation level (0-9)

Available at all Geant4 states.

Parameters
Z type i
A type i
Q type i Omittable : default value = 0
Lvl type i Omittable : default value = 0.0

time [t0] [Unit]

Set initial time of the particle.

Available at all Geant4 states.

Parameters
t0 type d
Unit type s Omittable : default value = ns Parameter candidates : s ms us ns ps min h d y second millisecond microsecond nanosecond picosecond minute hour day year

Set number of particles to be generated per vertex.

Range : N>0

Available at all Geant4 states.

Parameters
N type i

verbose [level]

Set Verbose level for GPS
0 : Silent
1 : Limited information
2 : Detailed information

Range : level>=0 && level <=2

Available at all Geant4 states.

Parameters
level type i
