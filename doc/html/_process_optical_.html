processActivation [proc_name] [flag]

Activate/deactivate the specified optical process

Available Geant4 state(s) : PreInit

Parameters
proc_name type s Parameter candidates : Cerenkov Scintillation OpAbsorption OpRayleigh OpMieHG OpBoundary OpWLS OpWLS2
flag type b Omittable : default value = 1

verbose [ver]

Set default verbose level for optical processes

Range : ver>=0

Available Geant4 state(s) : PreInit Idle

Parameters
ver type i Omittable : default value = 1

Print all optical parameters.
