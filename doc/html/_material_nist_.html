/material/nist/

Commands for the nist dataBase

Commands

printElement
printElementZ
listMaterials

printElement [symbol]

print element(s) in dataBase.
symbol = element.
all = all elements.

Available at all Geant4 states.

Parameters
symbol type s Omittable : default value = all

printElementZ [Z]

print element Z in dataBase.
0 = all elements.

Range : 0<=Z && Z<108

Available at all Geant4 states.

Parameters
Z type i Omittable : default value = 0

listMaterials [matlist]

Materials in Geant4 dataBase.
simple - simple NIST materials.
compound - compound NIST materials.
hep - HEP materials.
bio - biomedical materials.
all - list of all Geant4 materials.

Available at all Geant4 states.

Parameters
matlist type s Omittable : default value = all
