type [AngDis]

Sets angular source distribution type
Possible variables are: iso, cos, planar, beam1d, beam2d, focused or user

Available at all Geant4 states.

Parameters
AngDis type s Parameter candidates : iso cos planar beam1d beam2d focused user

rot1 [AR1x] [AR1y] [AR1z]

Sets the 1st vector for angular distribution rotation matrix
Need not be a unit vector

Range : AR1x != 0 || AR1y != 0 || AR1z != 0

Available at all Geant4 states.

Parameters
AR1x type d
AR1y type d
AR1z type d

rot2 [AR2x] [AR2y] [AR2z]
