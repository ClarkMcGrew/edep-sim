opacity [opacity]

Set the log of the relative opacity. Useful values are between [-0.5, 0.5], and +/-10 makes the object opague or transparent. A value of 0 leaves the opacity unchanged.

Available at all Geant4 states.

Parameters
opacity type d

daughterOpacity [opacity]

Set the log of the relative opacity of the daughters.

Available at all Geant4 states.

Parameters
opacity type d

check [check]

If this is true then check for overlaps
