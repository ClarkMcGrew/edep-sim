type [DisType]

Sets source distribution type.
Either Point, Beam, Plane, Surface or Volume

Available at all Geant4 states.

Parameters
DisType type s Parameter candidates : Point Beam Plane Surface Volume

shape [Shape]

Sets source shape for Plan, Surface or Volume type source.

Available at all Geant4 states.

Parameters
Shape type s Parameter candidates : Circle Annulus Ellipse Square Rectangle Sphere Ellipsoid Cylinder EllipticCylinder Para

Available at all Geant4 states.

Parameters
X type d
Y type d
Z type d
Unit type s Omittable : default value = cm Parameter candidates : pc km m cm mm um nm Ang fm parsec kilometer meter centimeter millimeter micrometer nanometer angstrom fermi

rot1 [R1x] [R1y] [R1z]

Set the 1st vector defining the rotation matrix'.
It does not need to be a unit vector.

Range : R1x != 0 || R1y != 0 || R1z != 0

Available at all Geant4 states.

Parameters

R1y type d
R1z type d
