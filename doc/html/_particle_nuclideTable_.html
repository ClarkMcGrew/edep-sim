Nuclide table control commands.

Commands

min_halflife

min_halflife [life] [Unit]

Set threshold of half-life.
Unit of the time can be :
s, ms, ns (default)

Range : halflife > 0.0

Available Geant4 state(s) : PreInit Idle

Parameters
life type d
Unit type s Omittable : default value = ns Parameter candidates : s ms us ns ps min h d y second millisecond microsecond nanosecond picosecond minute hour day year
