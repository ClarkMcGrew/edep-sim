Set the input file.

Available at all Geant4 states.

Parameters
name type s

tree [tree]

Set the tree path in the input file.

Available at all Geant4 states.

Parameters
tree type s

generator [generator]

Set the name of the kinematics source.

Available at all Geant4 states.

Parameters
generator type s

order [order]

Set order that events in the file are used.

Available at all Geant4 states.

Parameters
order type s Parameter candidates : consecutive stride random

number type i
